Hugo的使用及常用命令

2020年7月30日
| 阅读

hugo常用命令

01 hugo new post/**.md

表示新创建一篇文章在post文件夹里

02 hugo 编译生成静态文件 Hugo将编译所有文件并输出到public目录

03 hugo server 编译生成静态文件并启动web服务

每一次在根目录202007blog里更新文章———删除public文件——hugo（更新最新public文件夹）——上传public文件夹到github
